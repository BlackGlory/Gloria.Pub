https://gloria.pub/

A website to share your Gloria task.

## Start up

General:

```sh
git clone https://github.com/BlackGlory/Gloria.Pub.git
cd Gloria.Pub
npm install
npm run dev
```

Docker:

```sh
docker run -it blackglory/gloria.pub
```
